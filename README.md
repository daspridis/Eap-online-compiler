# EAP Pseudocode Interpreter
Ένας απλής, self-contained, in-browser Interpreter για τον ψευδοκώδικα που διδάσκεται στο πλαίσιο της ενότητας ΠΛΗ10 του Ελληνικού Ανοικτού Πανεπιστημίου (ΕΑΠ).

**Περιγραφή**

Αυτό το project παρέχει ένα ελαφρύ περιβάλλον για την εκτέλεση ψευδοκώδικα απευθείας στον browser, χωρίς την ανάγκη εγκατάστασης άλλου λογισμικού. Η εφαρμογή είναι client-side (JavaScript) και υλοποιεί tokenizer, parser και interpreter ώστε ο κώδικας να εκτελείται τοπικά στον browser.

**Χαρακτηριστικά**

- Υποστήριξη βασικής δομής: `ΑΛΓΟΡΙΘΜΟΣ`, `ΑΡΧΗ`, `ΤΕΛΟΣ`.
- Δηλώσεις `ΣΤΑΘΕΡΕΣ` και `ΔΕΔΟΜΕΝΑ`, συμπεριλαμβανομένων πινάκων (`ARRAY`) και πολυδιάστατων δομών.
- Εντολές ανάθεσης με `:=`.
- Έλεγχος ροής: `ΕΑΝ ... ΤΟΤΕ ... ΑΛΛΙΩΣ ... ΕΑΝ-ΤΕΛΟΣ`, βρόχοι `ΓΙΑ`, `ΕΠΑΝΑΛΑΒΕ`, `ΕΝΟΣΩ` κ.ά.
- Εντολές εισόδου/εξόδου: `ΤΥΠΩΣΕ(...)`, `ΔΙΑΒΑΣΕ(...)` (prompt για εισαγωγή από χρήστη).
- Υποπρογράμματα: `ΔΙΑΔΙΚΑΣΙΑ` και `ΣΥΝΑΡΤΗΣΗ`, με παράδοση παραμέτρων by value και by reference (σύμβολο `%`).
- Πλήρης υποστήριξη εκφράσεων (αριθμητικών, συγκριτικών, λογικών) με σωστή προτεραιότητα.

**Δομή Αρχείων**

- `frontend/`
  - `index.html` : κύρια σελίδα (UI)
  - `app.js` : λογική διεπαφής και χειρισμός UI
  - `core.js` : tokenizer / parser / interpreter (κεντρικός interpreter)
  - `style.css` : βασικά στυλ
- `tests/`
  - `interpreter_test.js` : δοκιμαστικά scripts για τον interpreter

**Πώς να το χρησιμοποιήσετε**

1. Ανοίξτε το `frontend/index.html` σε έναν σύγχρονο browser (Chrome, Firefox, Edge). Μπορείτε απλά να κάνετε διπλό κλικ στο αρχείο ή να το ανοίξετε με το μενού του browser.

2. Εναλλακτικά, εκκινήστε έναν απλό static server και επισκεφθείτε `http://localhost:8000`.
   - Με Python (PowerShell): `python -m http.server 8000 --directory frontend`
   - Με Node (`http-server`): `npx http-server frontend -p 8000`

3. Επικολλήστε ή γράψτε τον ψευδοκώδικά σας στο αριστερό πάνελ, πατήστε `Εκτέλεση` και δείτε το αποτέλεσμα/μηνύματα στην έξοδο.

**Τρέξιμο Δοκιμών**

Αν θέλετε να εκτελέσετε το `tests/interpreter_test.js` τοπικά με Node.js:

- Εκτέλεση απευθείας: `node .\tests\interpreter_test.js`
- Με Mocha: εγκατάσταση `npm install --save-dev mocha` και εκτέλεση `npx mocha tests`

Σημείωση: Οι δοκιμές μπορεί να απαιτούν επιπλέον setup ή εξαρτήσεις ανάλογα με το περιεχόμενό τους.

**Τεχνολογίες**

- HTML5, CSS3
- Vanilla JavaScript (ES6+)

Ολόκληρος ο tokenizer, ο parser και ο interpreter έχουν γραφτεί από την αρχή σε JavaScript και εκτελούνται τοπικά στον browser.
