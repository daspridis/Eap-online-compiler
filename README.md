# EAP Pseudocode Interpreter

Ένας απλός, self-contained, in-browser Interpreter για τον ψευδοκώδικα που διδάσκεται στο πλαίσιο της ενότητας ΠΛΗ10 του Ελληνικού Ανοικτού Πανεπιστημίου (ΕΑΠ).

## Περιγραφή

Αυτό το project δημιουργήθηκε για να παρέχει ένα αξιόπιστο και εύχρηστο περιβάλλον για την εκτέλεση του ψευδοκώδικα του ΕΑΠ, απευθείας στον browser, χωρίς την ανάγκη εγκατάστασης λογισμικού ή χρήσης command-line tools.

Ολόκληρη η εφαρμογή περιέχεται σε ένα μόνο αρχείο HTML, κάνοντας τη χρήση και τη διανομή της εξαιρετικά απλή.

## Χαρακτηριστικά

Ο Interpreter υποστηρίζει το μεγαλύτερο μέρος της σύνταξης του ψευδοκώδικα, συμπεριλαμβανομένων:

- **Βασική Δομή:** `ΑΛΓΟΡΙΘΜΟΣ`, `ΑΡΧΗ`, `ΤΕΛΟΣ`.
- **Δηλώσεις:** `ΣΤΑΘΕΡΕΣ` και `ΔΕΔΟΜΕΝΑ` (συμπεριλαμβανομένων πολυδιάστατων πινάκων `ARRAY`).
- **Εντολές Ανάθεσης:** `:=`.
- **Εντολές Ελέγχου Ροής:**
  - `ΕΑΝ ... ΤΟΤΕ ... ΑΛΛΙΩΣ ... ΕΑΝ-ΤΕΛΟΣ`
  - `ΓΙΑ ... ΕΩΣ ... ΜΕ ΒΗΜΑ ... ΕΠΑΝΑΛΑΒΕ ... ΓΙΑ-ΤΕΛΟΣ`
  - `ΕΝΟΣΩ ... ΕΠΑΝΑΛΑΒΕ ... ΕΝΟΣΩ-ΤΕΛΟΣ`
  - `ΕΠΑΝΑΛΑΒΕ ... ΜΕΧΡΙ ...`
- **Είσοδος / Έξοδος:**
  - `ΤΥΠΩΣΕ(...)` για την εμφάνιση αποτελεσμάτων.
  - `ΔΙΑΒΑΣΕ(...)` που ανοίγει ένα παράθυρο `prompt` για την εισαγωγή δεδομένων από τον χρήστη.
- **Υποπρογράμματα:**
  - `ΔΙΑΔΙΚΑΣΙΑ` και `ΣΥΝΑΡΤΗΣΗ` (δήλωση και κλήση).
  - Πέρασμα παραμέτρων κατ' αξία (by value) και κατ' αναφορά (by reference με το σύμβολο `%`).
- **Εκφράσεις:** Πλήρης υποστήριξη αριθμητικών, συγκριτικών και λογικών τελεστών με σωστή προτεραιότητα.

## Πώς να το χρησιμοποιήσετε

Η χρήση της εφαρμογής είναι πολύ απλή:

1.  Ανοίξτε το αρχείο `index.html` που βρίσκεται στον φάκελο frontend σε οποιονδήποτε σύγχρονο web browser (π.χ., Chrome, Firefox, Edge, Brave) σε ΟΠΟΙΑΔΗΠΟΤΕ συσκευή σας.
2.  Γράψτε ή επικολλήστε τον ψευδοκώδικά σας στο αριστερό παράθυρο ("Κώδικας Ψευδογλώσσας").
3.  Πατήστε το κουμπί **"Εκτέλεση"**.
4.  Το αποτέλεσμα της εκτέλεσης (ή τυχόν σφάλματα) θα εμφανιστεί στο δεξί παράθυρο ("Αποτέλεσμα").

## Τεχνολογίες

-   **HTML5**
-   **CSS3**
-   **Vanilla JavaScript (ES6+)**

Ολόκληρος ο tokenizer, ο parser, και ο interpreter έχουν γραφτεί από την αρχή σε JavaScript και εκτελούνται τοπικά στον browser σας.
